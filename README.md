# easyHATS

### HATS pack with minimal software - No more unnecessary homebrew - No more atrocious customization
I have to mantain several consoles and their users want something easy, fast, clean and pretty. I found this is the easiest way to keep them updated. They don´t care about fancy homebrew or amazing features. They just want it to WORK. And me too :)

Included programs:
- Atmosphere.
- Hekate.
- AIO Switch Updater.
- Quick Reboot app.
- Sphaira.
- Switch 90DNS Tester.
- MissionControl.
- Sigpatches.
- HB App Store.
- Tinfoil.
- nx-ovlloader.
- Tesla-Menu.
- QuickNTP.

Additional features:
- DNS.MITM.
- Exosphere.
- Custom hekate_ipl.ini (just changing some names, no big deal).
